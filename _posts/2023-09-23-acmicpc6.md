# 10818 최소, 최대

## sort

```c++
#include <iostream>
#include <algorithm>
using namespace std;

int arr[1000004];
int n;

int main() {

	ios_base::sync_with_stdio(0); cin.tie(0); cout.tie(0);

	cin >> n;
	for (int i = 0; i < n; i++)
		cin >> arr[i];
	sort(arr, arr + n);
	cout << arr[0] << " " << arr[n - 1] << "\n";
	return 0;
}
```
